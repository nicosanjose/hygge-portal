# Portal Hygge Services Mallorca · demo interactiva

Demo comercial del portal: **la misma aplicación que el producto real** (misma interfaz, botones y funciones) con la capa de datos sustituida por datos de ejemplo en memoria (`js/demo.js`), sin backend.

- Entrar como **Dirección** o como **Equipo (app de Cati)** desde la pantalla de acceso, o cambiar de vista desde el menú de usuario.
- Todo es interactivo: crear propiedades y reservas, planificar, fichar con foto, checklist, incidencias, compras, ausencias, informes y facturas. Los cambios viven solo en la sesión del navegador.
- El producto real (con Supabase) está en [hygge-app](https://github.com/nicosanjose/hygge-app).
